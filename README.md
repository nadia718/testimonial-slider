# testimonial-slider
A responsive and interactive Testimonial Slider built using HTML, CSS, and JavaScript. This project showcases customer reviews and testimonials with smooth sliding transitions.

## Features

* Smooth testimonial sliding effect
* Modern and clean user interface
* Interactive user experience
* Lightweight and beginner-friendly project

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

testimonial-slider/
│
├── index.html
├── style.css
└── script.js

## Project Goal

This project was created to practice JavaScript DOM manipulation, event handling, and slider functionality while building a common UI component used in modern websites.

## Functionality

* Display customer testimonials dynamically
* Navigate between testimonials
* Smooth transition animations
* Responsive layout for all screen sizes

Optimized for:
